==========================================
        Jogo da Memória - Pirata
==========================================
Sobre o jogo: Este é um jogo da memória em versão demo 2D.  
O objetivo é encontrar os pares corretos de cartas 
antes de cometer 9 erros.  

Se o jogador errar 9 vezes, o jogo exibe "GAME OVER"
e reinicia automaticamente.  
Se encontrar todos os pares, o jogador vence a partida.

Como jogar:

- Clique com o mouse em uma carta para revelá-la.
- Encontre o par correspondente para marcar ponto.
- Se errar 9 vezes GAME OVER.
- Pressione ESC para sair do jogo.
